# Character Assets

Place your character PNG layers here:

- `base.png` - Base character body
- `body.png` - Body layer
- `eye_base.png` - Neutral eyes
- `eye_happy.png` - Happy eyes (curved)
- `eye_angry.png` - Angry eyes
- `eye_sad.png` - Sad eyes
- `mouth_closed.png` - Closed mouth
- `mouth_open.png` - Open mouth (talking)
- `blush.png` - Blush effect (overlay)
- `accessories.png` - Optional accessories

Recommended size: 512x512px, PNG with transparency
